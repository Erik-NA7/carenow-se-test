# carenow-se-test
### Requires node>=20
## Vite React Frontend
```
cd frontend
```
### Scripts:
local development:
```
pnpm dev
or
npm run dev
// Run on port 3000
```
build:
```
pnpm build
or
npm run build
```
preview:
```
pnpm preview
or
npm run preview
// Run on port 3000
```


## Express Backend
From project root directory:
```
cd backend
```
### Scripts:
local development:
```
pnpm dev
or
npm run dev
// Run on port 3000
```
build:
```
pnpm build
or
npm run build
```
start:
```
pnpm start
or
npm start
// Run on default port 5000
```
